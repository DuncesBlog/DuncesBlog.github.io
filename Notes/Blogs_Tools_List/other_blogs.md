# 下面都是道听途说的

## CMD Markdown
    为记录思想和分享知识 提供更高效的工具,电脑手机都支持;
    除了云同步,离线功能这些基础功能外,还支持流程图、序列图、甘特图、表格等高级功能

## Typora
    一款Markdown编辑器,能把源码编辑和效果预览了两者合二为一,在编辑的时候即时生成预览效果

## GitBook
    用Git来支持文档类的写作,支持多人合作编辑,保证多人同时工作;并且保留所有的编辑历史,可以直接发布到github上
